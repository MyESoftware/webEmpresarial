# Agencia Web (Frontend)

React + Vite + Tailwind + React Router + SEO con react-helmet-async.
Listo para desplegar en Vercel.

## Ejecutar
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Personalización
- `src/config/site.js` (marca, WhatsApp, redes, email)
- `src/pages/Portfolio.jsx` (proyectos)
